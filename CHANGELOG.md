# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.0] - 2026-03-04

### Added
- Initial release of Mirra file server
- Directory listing with file icons
- Dark/Light theme support with system preference detection
- README.md rendering in directory listings
- Breadcrumb navigation
- Statistics display (directories, files, total size)
- Sortable columns (Name, Size, Modified)
- Latest modification time display
- Responsive design for mobile devices
